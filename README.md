# Linux File system — Notes & Learnings

## 🤯 The Biggest Realization
For the longest time, i believed that linux is an operating system just like macOS!! But turns out i was wrong!!! 

**"Linux is *not* an operating system. It is a kernel. It is the software that talks to hardware and orchestrates it to get the job done!! It is not the operating system. It is the part of an operating system**

And second big realization to me 

**Files as we know are something in which we store things like text, images, videos and others. But files in linux are not those. Everything that can give you data or accept data you give is a file. A keyboard is a file since you can GET data as to what is typed!**
